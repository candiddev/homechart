## Features

### Multiple Calendar > Event Reminders

Calendar > Events can now have multiple reminders before the event starts or ends.

## Enhancements

- Budget > Recurring Transactions can now have end dates.  Once an end date is reached, the Budget > Recurring Transaction will remain but not added.
- Recurring Shop > Items can now have end dates.  Once an end date is reached, the Shop > Item will be removed.
- Update Go to 1.25.2

## Fixes

- Fixed a bunch of issues with importing iCalendar ICS files and URLs.  Please let us know if there are any issues with import iCalendars!
